# Поиск и сравнение фильмов через API [poiskkino.dev](https://poiskkino.dev)

## клонируем репозиторий
git clone https://github.com/SergeyKapralov/internship.git

## переходим в директорию и делаем чистую установку зависимостей
cd internship
npm ci

## создаем файл .env в корне проекта и добавляем в него данные из env.default
## или ваш ключ с KP API записываем в VITE_API_KEY

## запускаем приложение
npm run dev режим разработчика 
npm run build cборка приложения

### структура проекта
src/
├── app/           # Инициализация приложения, роутинг, провайдеры
├── pages/         # Страницы приложения
├── widgets/       # Виджеты, изолированные сущности (каталог, сравнение)
├── features/      # Фичи, логика работы с сущностями (фильтры, действия с фильмами)
├── entities/      # Бизнес-сущности (фильмы, сравнение)
└── shared/        # Переиспользуемые компоненты, хуки, API, lib

#### Основные библиотеки
- `@tanstack/react-query` — кэширование и синхронизация данных с сервером
- `zustand` — глобальное состояние (сравнение, фильтры)
- `lucide-react` — иконки
- `react-router-dom` — навигация
- `tailwindcss` — утилитарные CSS классы
